# DigiturkMakaleApi

1-Projede kullanıdığınız tasarım desenleri hangileridir? Bu desenleri neden kullandınız?

Repository Design pattern kullandım. Tekrarlardan kurtulmak için bu deseni tercih ettim.

2-Kullandığınız teknoloji ve kütüphaneler hakkında daha önce tecrübeniz oldu mu? Tek tek
yazabilir misiniz?

Swagger ile wep api entegrasyonu haricinde bu projede kullandığım teknolojileri daha önceden kullanmıştım.Kullandığım teknolojiler;
Dependency Injection,
EntityFramework Core Code First,
AutoMapper,
.Net Core Web Api,
Repository Design Pattern,
ActionFilter,
Swagger


3-Daha geniş vaktiniz olsaydı projeye neler eklemek isterdiniz?

Adapter Design Pattern kullanarak NLog ile loglama işlemi yapmak isterdim.


4-Eklemek istediğiniz bir yorumunuz var mı? 

Proje için WebUI katmanı da hazırladım. İstenildiği takdirde onu da github a commitleyip aktarabilirim.
